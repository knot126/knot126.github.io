Smash Hit: Shatter for Blender

Shatter for Blender is a set of tools help users create custom levels using the Blender 3D modelling software as a base. It features a full mesh baker (e.g. part of a graphics pipeline) and miscellainous tools, like a mostly secure automatic updater.

Games and reverse engineering

I have been involved in reverse engineering video games. Addimttedly, I don't actually play games very much, since I like the game of tech. :P

By "reverse engineering", I mean I mostly work on tech, and not making new content for games. I think it's intresting to see how various games organise their code and solve problems.

By Mediocre AB

I have done a lot of things for Smash Hit, to the point where it has been my life for over three years. I am often active on the Smash Hit Lab discord server.

Smash Hit: Shatter

Shatter is a set of tools help users create custom levels using the Blender 3D modelling software as a base. It features a full mesh baker (e.g. part of a graphics pipeline) and many miscellainous tools, like a procedureal generator.

Smash Hit: Tweak Tool

Tool that allows patching libsmashhit.so to make tweaks to the game, like removing anti-tamper, changing feild of view and enabling file functions.

Smash Hit: Carambola

A failed attempt to write a level editor to replace Shatter for Blender (at the time called Smash Hit Blender Tools), written in Godot. Works for building basic stages.

All Games: Ad Server

A basic advertisement server implementation for mediocre games.

By Argonaut Games PLC

These games have the most intresting tech of any game listed here, mainly becuase most of it is custom. I have been mostly involved with the scripting language for the original Croc: Legend of the Gobbos (1997).

Croc Engine: Stratigise

A work in progress dissassembler written to help reverse the Argonaut Strategy Language (ASL) interpreter. There is also a reassembler, though it is a bit complex to get re-assembled things working.

By Ngmoco, LLC

I've only really done this to see what an old childhood game was like after its servers shut down over 10 years ago.

Touch Pets series: Katten

I am writing a small project to recreate the server software for Touch Pets: Cats. This also requires rewriting the social API for the long-dead Plus+ social gaming service.

Made with Marmalade SDK

More childhood games, but they are still active so I don't really care as much.

Marmalade Modding Tools

Some tools to assist in modding games made with the Marmalade SDK. Currently only contains a basic dz archive extractor.
